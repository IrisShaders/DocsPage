### `const float ambientOcclusionLevel = 1.0;`

**Location**: any GLSL file

---

Sets the amount of vanilla ambient occlusion that is applied through the vertex color [attribute](/current/reference/attributes/overview). The default value is 1.0 (vanilla level), and a value of 0.0 will disable vanilla ambient occlusion. The valid range for this value is [0.0..1.0].
