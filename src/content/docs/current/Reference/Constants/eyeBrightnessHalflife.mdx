### `const float eyeBrightnessHalflife = 10.0;`

**Location**: any GLSL file

---

Sets the half life in ticks for the transition time for the [`eyeBrightnessSmooth`](/current/reference/uniforms/camera#eyebrightnesssmooth) uniform. The default value is 10 ticks.
