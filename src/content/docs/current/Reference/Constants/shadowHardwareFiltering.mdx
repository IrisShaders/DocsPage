### `const bool shadowHardwareFiltering = true;`

**Location**: any GLSL file

---

Enables hardware shadow sampling (depth comparison mode) for [`shadowtex`](/current/reference/buffers/shadowtex) buffers. See [Hardware Sampling](/current/reference/buffers/shadowtex) for more info.
