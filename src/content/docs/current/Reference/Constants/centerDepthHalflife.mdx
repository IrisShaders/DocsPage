### `const float centerDepthHalflife = 1.0;`

#### Location: any glsl file

Sets the half life in ticks for the transition time for the [`centerDepthSmooth`](/current/reference/uniforms/camera#centerdepthsmooth) uniform. The default value is 1 tick.
