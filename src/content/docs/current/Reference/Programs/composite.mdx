**Required stages**: vertex, fragment  
**Optional stages**: geometry, compute  
**Output buffers**: [colortex](/current/reference/buffers/colortex)  
**Valid suffixes**: no suffix, 1-99

-------------------------------------------------

A composite-style pass which runs after all [gbuffers](/current/reference/programs/gbuffers) programs. Like other composite-style programs, it renders a single fullscreen quad over the resolution of the buffer(s) it writes to.
