**Required stages**: compute  
**Optional stages**: N/A  
**Valid suffixes**: no suffix, 1-99  

-------------------------------------------------

An Iris-exclusive compute pass. It is run during the pack load or when the screen size changes. Unlike other shader passes, setup only supports compute shaders.
