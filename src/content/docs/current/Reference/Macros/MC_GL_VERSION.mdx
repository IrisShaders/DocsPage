### `MC_GL_VERSION`

The maximum OpenGL version supported by the system, encoded in an integer format without the decimal point. This may be lower than the [maximum GLSL version](/current/reference/macros/mc_glsl_version) is it relates to the OpenGL version used by Minecraft.

For example:
- OpenGL 2.1 -> 210
- OpenGL 3.3 -> 330
- OpenGL 4.6 -> 460
