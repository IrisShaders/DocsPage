### `IRIS_VERSION`

The current Iris version, encoded in a 122 format (1 major, 2 minor, 2 release). This macro was added in Iris 1.8, and as such no earlier versions will be defined.

For example:
- 1.8.2 → 10802
- 1.8.10 → 10810
