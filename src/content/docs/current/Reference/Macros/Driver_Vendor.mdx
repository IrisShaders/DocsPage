title: Driver Vendor
description: The vendor of the user's GPU driver.
sidebar:

  order: 2
---

Based on the current GPU driver vendor, one of the following macros will be defined:

- `MC_GL_VENDOR_AMD`
- `MC_GL_VENDOR_ATI`
- `MC_GL_VENDOR_INTEL`
- `MC_GL_VENDOR_MESA`
- `MC_GL_VENDOR_NVIDIA`
- `MC_GL_VENDOR_XORG`
- `MC_GL_VENDOR_OTHER`
