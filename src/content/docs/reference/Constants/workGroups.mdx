#### `const ivec3 workGroups = ivec3(<groups_x>, <groups_y>, <groups_z>);`

#### Location: any compute shader

Used to define a fixed number of work groups in a compute shader. Replace `<groups_x>`, `<groups_y>`, and `<groups_z>` with the number of work groups in each dimension. For more info, see the [Compute Shader](/reference/programs/overview#compute-shaders) section.
