### `uniform int isEyeInWater;`

This uniform outputs the current substance that the camera is in. Possible values are:

| value | state         |
| ----- | ------------- |
| `0`   | none/other    |
| `1`   | water         |
| `2`   | lava          |
| `3`   | powdered snow |
