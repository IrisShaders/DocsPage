### `uniform vec3 upPosition;`
This uniform stores the up value in [view space](/how_to/spaces). This value can also be pulled from [`gbufferModelView[1].xyz`](/reference/uniforms/matrices/gbuffermodelview). The length of this adds up to 100 (`normalizedValue * 100`).
