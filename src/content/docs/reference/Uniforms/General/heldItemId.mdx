### `uniform int heldItemId;`

This uniform stores the id(`shader.properties`) of the item currently held in the main hand. Value is 0 if no `item.properties` is ProgressEvent. Value is -1 if `item.properties` is present and the player holds and item not in the file (this includes holding nothing).
