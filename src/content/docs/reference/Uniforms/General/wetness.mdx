### `uniform float wetness;`

This uniform is equivalent to [`rainStrength`](/reference/uniforms/general/rainstrength), except the transition time is controlled with [`wetnessHalfLife`](/reference/constants/wetnesshalflife). The default transition time is also significantly slower.
