### `uniform int entityId;`

This uniform stores the id (from [`entity.properties`](/reference/miscellaneous/entity_properties)) of the current entity. Value is `0` if no [`entity.properties`](/reference/miscellaneous/entity_properties) is present. Value `65535` if [`entity.properties`](/reference/miscellaneous/entity_properties) is present and the entity is not in the file.
