<Aside type="danger">This uniform only works on `1.17` and newer.</Aside>

### `uniform mat4 projectionMatrixInverse;`

Equal to `inverse(projectionMatrix)`, but calculated on the CPU for improved performance since the `inverse()` function is very slow.
