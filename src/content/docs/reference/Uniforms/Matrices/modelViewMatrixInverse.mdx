<Aside type="danger">This uniform only works on `1.17` and newer.</Aside>

### `uniform mat4 modelViewMatrixInverse;`

Equal to [`inverse(modelViewMatrix)`](/reference/uniforms/matrices/modelviewmatrix), but calculated on the CPU to avoid running the expensive `inverse()` function on the GPU.
